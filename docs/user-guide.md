# 利用者向けガイド

## 基本
1. 受け取った募集ボックスURLを開く
2. 指定された条件（形式/サイズ/件数/期限）を確認（サイズが「無制限」の場合あり）
3. 必要に応じてパスワード、送信者名、メモを入力
4. ファイルをアップロード

## 閲覧専用アカウント向け
管理者から配布された「閲覧アカウント」でログインすると、
- 割り当て済み募集ボックスのファイル一覧表示
- ファイルプレビュー
- ファイルダウンロード
が可能です。

> 閲覧アカウントは、募集ボックス作成や設定変更はできません。

## 追加カスタマイズ表示
ボックスによっては以下が設定されています。
- ヘッダー画像
- カスタムフォント
- 管理者の案内メッセージ
- 完了後の誘導URL

## 注意
- 停止中または期限切れのボックスには送信できません。
- 拡張子や容量制限はサーバー側で検証されます。


## 許可拡張子の入力補助（管理者向け）
- 募集ボックス作成/編集で「画像ファイル」「文書ファイル」などのプリセットを選ぶと、拡張子が一括入力されます。
- 候補付き入力欄に文字を入れると候補を選択でき、追加ボタン/Enterでタグ追加できます。
- 既存の手動入力（カンマ区切り）もそのまま利用できます。

## アップロード時のファイル一覧表示
- 複数ファイルを選択すると、送信前に選択済みファイル名とサイズが一覧表示されます。
