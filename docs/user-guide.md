# 利用者向けガイド

## 1. 概要
Uploader は、管理者が作成した「募集ボックス」に対して、リンクを知っている利用者がファイルをアップロードできる OSS です。

## 2. 利用者の使い方
1. 管理者から共有されたリンク（例: `/box/xxx`）を開く
2. 画面に表示された制限（許可拡張子/サイズ/件数）を確認
3. 必要な場合はボックスパスワードを入力
4. ファイルを選択してアップロード

## 3. 管理者の使い方
1. 初回は `/admin/register` で管理者アカウントを作成
2. `/admin` でログインして募集ボックスを作成
3. 共有リンクを配布
4. `/admin` からアップロード状況確認、ファイルダウンロード、ボックス停止/再開

## 4. 注意事項
- 拡張子とサイズ制限はサーバー側でも検証されます。
- Discord 通知は Webhook URL 設定時のみ送信されます。
- 不特定多数向け公開時は HTTPS 化を推奨します。
